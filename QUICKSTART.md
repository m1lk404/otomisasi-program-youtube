## First Download

1. **Copy a YouTube URL**
   - Example: `https://www.youtube.com/watch?v=dQw4w9WgXcQ`

2. **Paste in the GUI**
   - Click the window's URL field
   - Ctrl+V to paste

3. **Pick a folder** (or use default)
   - Default goes to `~/Downloads/YouTube`

4. **Click Start Download**
   - Wait for completion message

5. **Check your download folder** ðŸŽ‰

## Common Tasks

### Download with Best Quality

1. Select "Best" in Quality section
2. No other options needed
3. Click Start Download

### Cut a 5-minute clip

1. Check "Enable Clip Cutting"
2. Enter Start: `2` (minutes)
3. Enter End: `7` (minutes)
4. Click Start Download

### Save Login Info

1. Open Chrome/Firefox
2. Login to YouTube
3. In app, select "Chrome" or "Firefox" in Cookie section
4. Download will use your login

## Troubleshooting

**"yt-dlp not found"**

- Windows: Reinstall the .exe
- Linux: `pip install --user yt-dlp`

**Stuck on download**

- Click Cancel button
- Try again in 5 minutes

**Can't download certain video**

- Enable cookies (Chrome/Firefox option)
- Try VPN if region-blocked
- Check if video is public
